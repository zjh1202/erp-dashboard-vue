# GitHub 配置

## 仓库信息
- **仓库地址**: https://github.com/zjh1202/erp-dashboard-vue
- **SSH地址**: git@github.com:zjh1202/erp-dashboard-vue.git
- **HTTPS地址**: https://github.com/zjh1202/erp-dashboard-vue.git

## 推送方式
使用 HTTPS + Personal Access Token 方式推送

## Token 设置
首次使用需要编辑 `push.sh` 脚本，将 `YOUR_GITHUB_TOKEN_HERE` 替换为你的 GitHub Personal Access Token。

## 自动推送脚本
项目根目录有 `push.sh` 脚本，可快速推送更新：
```bash
cd /workspace/projects/manufacturing-dashboard
./push.sh "提交信息"
```

## 注意事项
- Token 仅保存在本地 push.sh 文件中，请勿上传到 Git
- 更新后只需运行 `./push.sh` 即可自动提交并推送
